# 2. 编程语言内容
## 2.1 介绍
这是一个对寄存器的操作语言，还有跳转的编程语言
### 2.1.1 寄存器
以$R_i$表示寄存器。
### 2.1.2 对寄存器操作的指令
$set\ x\  y$表示将寄存器$y$中的值赋值给寄存器$x$。
$up\ x$表示将寄存器$x$中的值加1。
$down\ x$表示将寄存器$x$中的值减1。
$in\ x$表示将一个输入赋值到寄存器$x$中。
$out\ x$表示将寄存器$x$中的值输出。
### 2.1.3 跳转指令
$ifpos\ x\ goto \ z$表示如果$R_x > 0$，则将程序执行跳转到指令$z$处。
$ifnng\ x\ goto \ z$表示如果$R_x \ge 0$，则将程序执行跳转到指令$z$处。
$ifeq\ x\ y\ goto \ z$表示如果$R_x = R_y$，则将程序执行跳转到指令$z$处。
$ifneq\ x\ y\ goto \ z$表示如果$R_x \ne R_y$，则将程序执行跳转到指令$z$处。
但是$z < \text{当前指令的序号}$。
